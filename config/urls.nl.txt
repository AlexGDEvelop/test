# Negative-класс для замера: типичный браузинг в Нидерландах (то, на что маскируемся).
# Микс по весу: новости (текст+картинки), магазины, форумы, лёгкие сервисы/погода.
# Запуск:  ... --urls config/urls.nl.txt
#
# Заметки:
# - Чем ближе к ТВОЕМУ реальному браузингу, тем честнее фон. Подмешай свои сайты.
# - Некоторые сайты через CONNECT-прокси + HTTP/1.1 могут редиректить/отдавать
#   мало — для ФОРМЫ трафика это ок (--max-bytes всё равно ограничивает чтение).
# - НЕ добавляй сюда свой сервер/VPS — это не фон.
# - google.com/youtube.com намеренно НЕ включены: флаки через прокси (HTTP/2,
#   consent), дают 20-сек зависания. Добавь сам, если правда нужны.
